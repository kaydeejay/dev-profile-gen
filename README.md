# dev-profile-gen
A node.js command-line application that generates a resume pdf based on the given github user profile

## Dependencies
Node, npm

## Installation Guide
Download the directory, run 'npm i' in a terminal window

## How to Use this Application
run 'node index.js' in your terminal window

This application will ask for your github username, and your preferred color, and generate a pdf resume
based on the given information.
